# Hybrid Human Color Scheme

## Current Theme: Beige, Olive Green & Black

### Primary Colors
- **Background**: `#FAF0DC` (Warm Beige)
- **Card Background**: `#FFFFFF` (White)
- **Secondary Background**: `#FFF5E6` (Light Beige)
- **Tertiary Background**: `#F0E6D0` (Pale Beige)

### Accent Colors
- **Primary Accent**: `#556B2F` (Dark Olive Green)
- **Secondary Accent**: `#8FBC8F` (Light Olive Green)
- **Alert**: `#FF6B35` (Orange Red)
- **Info**: `#4ECDC4` (Teal)
- **Warning**: `#FFD700` (Gold)

### Text Colors
- **Primary Text**: `#1A1A1A` (Near Black)
- **Secondary Text**: `#4A4A4A` (Dark Gray)
- **Tertiary Text**: `#888` (Medium Gray)
- **Disabled Text**: `#CCC` (Light Gray)
- **White Text**: `#FFF` (For dark backgrounds)

### Border Colors
- **Primary Border**: `#D0C5B0` (Beige Gray)
- **Secondary Border**: `#E0D5C0` (Light Beige Gray)

### Usage Guidelines
- Use `#FAF0DC` for main container backgrounds
- Use `#FFFFFF` for card/content backgrounds
- Use olive green (`#556B2F`, `#8FBC8F`) for buttons, icons, and interactive elements
- Use black/dark gray (`#1A1A1A`, `#4A4A4A`) for all text
- Maintain sufficient contrast for accessibility
